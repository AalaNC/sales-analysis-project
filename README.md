# 📊 Análisis de Ventas y Rentabilidad

## 🧠 Descripción del proyecto

Este proyecto consiste en el análisis de un dataset de ventas de una empresa de retail con el objetivo de evaluar su desempeño, rentabilidad y detectar oportunidades de mejora en el negocio.

Se simula un caso real de trabajo de un analista de datos, aplicando limpieza, análisis y visualización de datos para obtener insights accionables.

---

## 🎯 Objetivos

* Analizar las ventas y la rentabilidad general del negocio
* Identificar las categorías y productos más rentables
* Detectar productos con pérdidas
* Evaluar el impacto de los descuentos en la rentabilidad
* Analizar el desempeño por región

---

## 🛠️ Herramientas utilizadas

* SQL (limpieza y análisis de datos)
* Tableau (visualización y dashboard)
* Excel (preparación del dataset)

---

## 📂 Dataset

* Fuente: Superstore Sales Dataset
* Registros: ~10.000 filas
* Variables principales: ventas, ganancia, categoría, región, descuentos y fechas

---

## 🔍 Análisis realizado

### 📈 Desempeño general

* La empresa es rentable con un margen aproximado del 15%

### 🏆 Categorías

* La categoría Tecnología es la más rentable y la que genera mayores ingresos
* Suministros de oficina presenta alto volumen de ventas pero bajo margen de ganancia

### ⚠️ Problemas de rentabilidad

* Se identificaron productos con ganancias negativas a pesar de tener ventas
* Esto sugiere problemas en precios, costos o estrategia de descuentos

### 💸 Impacto de descuentos

* Descuentos superiores al 30% generan pérdidas
* Indica que el uso excesivo de descuentos afecta negativamente la rentabilidad

### 🌍 Análisis por región

* El negocio se encuentra relativamente equilibrado entre regiones
* La región Oeste (West) es la más rentable

## 💡 Insights clave

* La rentabilidad depende principalmente de la categoría Tecnología
* Los descuentos elevados reducen significativamente las ganancias
* Existen productos que deberían ser revisados o eliminados
* El negocio es estable, pero tiene oportunidades claras de optimización

---

## 🚀 Conclusión

La empresa presenta un buen nivel de rentabilidad general, pero existen oportunidades de mejora en la estrategia de precios y descuentos, así como en la gestión de productos no rentables.
